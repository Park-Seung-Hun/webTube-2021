### 📕 주요 기능 

```
├── assets
     ├── js
     |   └── main.js
     └── scss   
         ├── config
         |   ├── _variables.scss
         |   └── reset.scss
         ├── pages
         |   └── home.scss
         └── partials
             ├── main.scss
             └── style.scss
             
└── static
     ├── main.js
     └── styles.css
```
```
├── README.md
├── SUMMARY.md
├── chapter-1/

    ├── README.md

```  

### 📘 추가할 기능
 - `달력` 기능 추가 
   1. 기본 달력틀을 JS와 CSS를 통해 구현한다.
   2. 달력에 기념일 & 할일 추가 기능을 JS를 통해 구현한다.

 - `Music` 기능 추가
   1. 날씨 or 계절에 맞는 음악 추천
   2. 앨범 표지 회전 효과
